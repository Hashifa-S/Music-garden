**MUSIC CARD PLAYER**

A simple web app that lets you create and share a custom "music card".

Users can upload an image, place it onto a CD-style player, add some visual elements, and play a song. Each card can be shared using a link for each custom card.

-----------------------------------------------------------------------------------------------------------------------------------------------------------
**Features**
- Upload an image and display it on a CD
- Basic music playback
- Decorative elements like flowers
- Shareable links for each custom card.

------------------------------------------------------------------------------------------------------------------------------------------------------------
**Tech Stack**
- HTML
- CSS
- JavaScript
- Music API for song search

------------------------------------------------------------------------------------------------------------------------------------------------------------
**How to run**

Open index.html in a browser.

To host it:
* Push the project to Github
* Go to repository settings
* Open the pages section
* Select the main branch

Github will generate a public link for the site

------------------------------------------------------------------------------------------------------------------------------------------------------------
**How it works**
- The user uploads an image - it's rendered onto the CD
- The player handles audio playback
- UI elements (like flowers) are layered using CSS
- Data can be stored in the URL or browser storage for sharing.

------------------------------------------------------------------------------------------------------------------------------------------------------------
**Future Improvements**
- Drag and position elements
- Save multiple cards
- Better Mobile Layout
- Better Animation ( Spinning CD, transitions)
------------------------------------------------------------------------------------------------------------------------------------------------------------
**Notes**

This project is meant to be lightweight and customizable. Most of the visuals can be swapped out by replacing images in the /images folder or adjusting CSS.
